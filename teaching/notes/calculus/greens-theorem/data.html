Section 6.4: Green’s Theorem

This section is an interesting one — in many multivariable calculus courses, it becomes something of a transitional fossil, since results we develop in later sections will supersede this one. However, I’d like to make the case that Green’s Theorem not only stands on its own as a useful theorem, but is also one of the most conceptually valuable results we’ll see in calculus: it’s the least complex version of a series of results we’ll develop over the remaining sections, and that gives it the distinction of being as simple as possible to learn and remember while still containing nearly all of the conceptual nuance we’ll see in the future sections’ theorems.

Green’s Theorem

We spent the last section discussing how curve integrals of a conservative vector field (dotted with the unit tangent vector of the curve) satisfy a theorem that is functionally the Fundamental Theorem of Calculus, but adapted to the world of multivariable calculus like many results have been before them. If there’s one important lesson we can learn from that theorem, it’s that while vector fields might seem like auxiliary objects that happen to be compatible with the language of curve integrals, they’re in fact a core part of the story.

Let’s return to the world of double integrals for a moment. When we integrate some function $g(x, y)$ over the region $R$ bounded by $y = \sqrt{x}$, $y = 0$, and $x = 1$, then the double integral

$$\begin{align*}\iint_D h(x, y)\,\mathrm{d} A = \int_0^1 \int_0^{\sqrt{x}} g(x, y)\,\mathrm{d} y\,\mathrm{d} x\end{align*}$$

is slicing the region vertically, with each slice measuring the area of a cross-section parallel to the $y$-axis.

Let’s focus on how we actually compute that integral: by finding an antiderivative of $g(x, y)$ with respect to $y$. To avoid needing to introduce extra notation for the antiderivative of $g$, let’s just suppose $g = \frac{\partial f}{\partial y} = f_y$ for some function $f(x, y)$. If that’s the case, then $f$ is the antiderivative we’re looking for, and

$$\begin{align*}\int_0^1 \int_0^{\sqrt{x}} f_y(x, y)\,\mathrm{d} y\,\mathrm{d} x &= \int_0^1 \left( f\left( x, \sqrt{x} \right) - f\left( x, 0 \right) \right) \,\mathrm{d} x\\[4px]&= \int_0^1 f\left( x, \sqrt{x} \right) \,\mathrm{d} x - \int_0^1 f\left( x, 0 \right) \,\mathrm{d} x.\end{align*}$$

In section 6.1, we only considered the inner integral as a single object measuring the area of a cross-section, but here we’re treating the inner integral as a difference of two functions in its own right. And now that we have the technology of curve integrals, both of those integrals above certainly look familiar! Each is a curve integral, but post-parameterization. Let’s swap out $x$ for $t$ to make it a bit more clear, and let’s also just focus on the first integral for now:

$$\begin{align*}\int_0^1 f\left( t, \sqrt{t} \right) \,\mathrm{d} t.\end{align*}$$

This looks very nearly like a curve integral of $f$ over the path $C$ given by $\vec{r}(t) = \left< t, \sqrt{t} \right>$, but there’s an unfortunate issue — the curve integral would have an extra factor of $\left| \left| \vec{r}\hspace{0.1em}'(t) \right| \right|$ when we parameterize:

$$\begin{align*}\int_C f\,\mathrm{d} s = \int_0^1 f\left( \vec{r}(t) \right) \left| \left| \vec{r}\hspace{0.1em}'(t) \right| \right|\, \mathrm{d} t.\end{align*}$$

That factor is more than a minor annoyance — it’s fundamentally changing the integral. Here, the derivative is equal to

$$\begin{align*}\vec{r}\hspace{0.1em}'(t) &= \left< 1, \frac{1}{2}t^{-1/2} \right>,\end{align*}$$

and depending on the function bounding the region, that second component could be arbitrarily bad. And yet by writing it out like this, we can see the faintest outline of a loophole. The first component will always be $1$ regardless of the upper bound on the region, since we’re parameterizing the graph of a function. When it’s packaged up along with the second component into $\left| \left| \vec{r}\hspace{0.1em}'(t) \right| \right|$, all of the complexity is back, but if we can work just with $\vec{r}\hspace{0.1em}'(t)$ and somehow only interface with its first component, we’ll be able to completely sidestep the potentially nasty second component and recover the actual integral we’re trying for.

The first of these two goals can be accomplished by switching to the other type of curve integral we developed in the last section: one of a vector field instead of a function of two variables. The integral

$$\begin{align*}\int_C \vec{F} \bullet \mathrm{d} \vec{r} = \int_0^1 \vec{F}(\vec{r}(t)) \bullet \vec{r}\hspace{0.1em}'(t)\, \mathrm{d} t\end{align*}$$

deals with the two components of $\vec{r}\hspace{0.1em}'(t)$ separately, and so if we integrate a field $\vec{F}$ whose second component is zero, then we can ignore the $y$-component of $\vec{r}\hspace{0.1em}'(t)$.

What about the first component? Our goal was for this integral to equal $\int_0^1 f\left( t, \sqrt{t} \right) \,\mathrm{d} t$, and if we make the first component of $\vec{F}$ equal to $f(x, y)$, we get just that:

$$\begin{align*}\int_C \vec{F} \bullet \mathrm{d} \vec{r} &= \int_C \left< f(x, y), 0 \right> \bullet \mathrm{d} \vec{r}\\[4px]&= \int_0^1 \left< f(\vec{r}(t)), 0 \right> \bullet \vec{r}\hspace{0.1em}'(t)\,\mathrm{d} t\\[4px]&= \int_0^1 \left< f(t, \sqrt{t}), 0 \right> \bullet \left< 1, \frac{1}{2}t^{-1/2} \right>\,\mathrm{d} t\\[4px]&= \int_0^1 f(t, \sqrt{t}) \cdot 1 \,\mathrm{d} t.\end{align*}$$

We can see that this worked algebraically, but why? The geometric explanation is that by placing $f(x, y)$ as the $x$-component of a vector field instead of integrating it directly, we removed out the issue of the bases of the rectangles in the curve integral’s Riemann sum growing and shrinking based on arc length. With this parameterization, they only do that in the $y$-direction, and so all of that variability is affecting the zero component of $\vec{F}$.

Now we can return to the bottom integral: even though the $y$-value is always zero, so it doesn’t take any extra work to express it as a curve integral, we can write it in terms of a vector field regardless in order to help this work generalize. If we label the side of the region $R$ given by $y = \sqrt{x}$ as $C_1$ and the side given by $y = 0$ as $C_2$, then we have

$$\begin{align*}\iint_D f_y(x, y)\,\mathrm{d} A &= \int_{C_1} \left< f(x, y), 0 \right> \bullet \mathrm{d} \vec{r} - \int_{C_2} \left< f(x, y), 0 \right> \bullet \mathrm{d} \vec{r}.\end{align*}$$
