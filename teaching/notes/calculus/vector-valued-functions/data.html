Section 5.3: Vector-Valued Functions

After getting our bearings in 3D space and graphing objects like lines, planes, and spheres, let’s start dipping our toes into graphing more complicated functions. We’ve already seen that the difference in dimension between lines and planes in $\mathbb{R}^3$ corresponds to a stark difference in how their equations are presented: planes are given by an equation, like $z = x + y$, while lines typically need to be parameterized as functions of the form $\vec{l}(t)$, taking in a single parameter $t$ and outputting a position vector in $\mathbb{R}^3$. The former is effectively a function that takes in two variables and outputs one — $z = f(x, y)$ — while the latter is a function that takes in one variable and outputs three. We’ll focus on that second case, called vector-valued functions, for now. Later, we’ll return to explore the former, called functions of multiple variables. Finally, the ending of the course will focus on functions that both take in and output more than one variable.

A general vector-valued function in $\mathbb{R}^3$ is of the form $\vec{r}(t) = \left< x(t), y(t), z(t) \right>$; we call $x(t)$, $y(t)$, and $z(t)$ its component functions. Just like with lines in $\mathbb{R}^3$, a vector-valued function traces out a path in $\mathbb{R}^3$ that we can think of as following the tip of a moving position vector.

Exercise: the shape of a vector-valued function

What shape is the graph of $\vec{r}(t) = \left< \cos(t), \sin(t), 1 \right>$? What about $\vec{s}(t) = \left< \cos(t), \sin(t), t \right>$?

The former is the unit circle in the plane $z = 1$. The latter is essentially the same, but it is continually moving up through $z$-planes. The result is a helix!

One place vector-valued functions arise is in the intersection of two surfaces — two different two-dimensional surfaces typically intersect in a one-dimensional curve.

Example: parameterizing a surface intersection

Find a vector-valued function $\vec{r}(t)$ for the intersection between $x^2 + y^2 + z^2 = 9$ and $z = x + 4$.

This is a plane cutting a sphere at an oblique angle, which produces a circle.

As if we were solving for an intersection of two curves in $\mathbb{R}^2$, we can start by plugging one of the formulas into the other. We have

$$\begin{align*}x^2 + y^2 + (x + 4)^2 &= 9\\[4px]2x^2 + 8x + 16 + y^2 &= 9\\[4px]x^2 + 4x + 8 + \frac{1}{2}y^2 &= \frac{9}{2}\\[4px]x^2 + 4x + 4 + \frac{1}{2}y^2 &= \frac{1}{2}\\[4px]\left( x + 2 \right)^2 + \frac{1}{2}y^2 &= \frac{1}{2}\\[4px]2\left( x + 2 \right)^2 + y^2 &= 1\\[4px]\left( \sqrt{2}(x + 2) \right)^2 + y^2 &= 1\end{align*}$$

The result is the equation of an ellipse in $\mathbb{R}^2$, which we can parameterize as

$$\begin{align*}\left( -2 + \frac{1}{\sqrt{2}} \cos(t), \sin(t) \right).\end{align*}$$

This is effectively the shadow that the red circle casts on the $xy$-plane. To lift it up to where it lives in $\mathbb{R}^3$, we just need to put back in the $z$-coordinate, which is $x + 4$:

$$\begin{align*}\vec{r}(t) = \left< -2 + \frac{1}{\sqrt{2}} \cos(t), \sin(t), 2 + \frac{1}{\sqrt{2}} \cos(t) \right>.\end{align*}$$

The components of vector-valued functions are largely independent from one another, and so limits of them are mercifully simple. If $x(t)$, $y(t)$, and $z(t)$ are approaching values $x_0$, $y_0$, and $z_0$ respectively as $t \to a$, then $\vec{r}(t) = \left< x(t), y(t), z(t) \right>$ is approaching $\left< x_0, y_0, z_0 \right>$. That is,

Proposition: limits of vector-valued functions

Let $\vec{r}(t) = \left< x(t), y(t), z(t) \right>$. Then

$$\begin{align*}\lim_{t \to a} \vec{r}(t) = \left< \lim_{t \to a} x(t), \lim_{t \to a} y(t), \lim_{t \to a} z(t) \right>,\end{align*}$$

assuming those limits exist. Consequently, $\vec{r}(t)$ is continuous at a value of $t$ if and only if its component functions are continuous there.

As in calculus 1, our next target is derivatives! The limit definition is our friend here: for a $\vec{r}(t) = \left< x(t), y(t), z(t) \right>$, its derivative is defined to be

$$\begin{align*}\vec{r}'(t) &= \lim_{h \to 0} \frac{\vec{r}(t + h) - \vec{r}(t)}{h}\\[4px]&= \lim_{h \to 0} \frac{\left< x(t + h),\ y(t + h),\ z(t + h) \right> - \left< x(t),\ y(t),\ z(t) \right>}{h}\\[4px]&= \lim_{h \to 0} \frac{\left< x(t + h) - x(t),\ y(t + h) - y(t),\ z(t + h) - z(t) \right>}{h}\\[4px]&= \lim_{h \to 0} \left< \frac{x(t + h) - x(t)}{h},\ \frac{y(t + h) - y(t)}{h},\ \frac{z(t + h) - z(t)}{h} \right>\\[4px]&= \left< \lim_{h \to 0} \frac{x(t + h) - x(t)}{h},\ \lim_{h \to 0} \frac{y(t + h) - y(t)}{h},\ \lim_{h \to 0} \frac{z(t + h) - z(t)}{h} \right>\\[4px]&= \left< x'(t), y'(t), z'(t) \right>,\end{align*}$$

assuming those exist.

What a ride! We used just about every property of vectors and vector-valued functions we’ve established so far, and the result is wonderfully simple.

We can use the fact that the derivative vector is tangent to the curve to find a formula for a tangent line, but a slight annoyance with the derivative is that its magnitude can vary substantially. To address that problem and make many future formulas more convenient, we define a magnitude-normalized version:

Definition: unit tangent vector

Let $\vec{r}(t)$ be a differentiable vector-valued function. The unit tangent vector to the graph of $\vec{r}(t)$ is

$$\begin{align*}T(t) = \frac{\vec{r}'(t)}{\left| \left| \vec{r}'(t) \right| \right|},\end{align*}$$

which is defined so long as $\vec{r}'(t) \neq \vec{0}$.

Since differentiation of a vector-valued function boils down to just differentiating the component functions, we don’t have many new derivative rules to write down. What we do have, though, are a few ways in which the derivative acts on operations we can perform on vectors. They’ll likely look familiar!

Proposition: vector-valued derivative rules

For any vector-valued functions $\vec{r}(t)$ and $\vec{s}(t)$,

$$\begin{align*}\frac{\text{d}}{\text{d}t} \left[ \vec{r}(t) \bullet \vec{s}(t) \right] &= \vec{r}'(t) \bullet \vec{s}(t) + \vec{r}(t) \bullet \vec{s}'(t)\\[4px]\frac{\text{d}}{\text{d}t} \left[ \vec{r}(t) \times \vec{s}(t) \right] &= \vec{r}'(t) \times \vec{s}(t) + \vec{r}(t) \times \vec{s}'(t)\end{align*}$$

In just the same way that derivatives of vector-valued functions are given by differentiating the components because limits split nicely into those components, we can integrate vector-valued functions component-wise too! Specifically,

$$\begin{align*}\int \vec{r}(t)\,\text{d} t = \left< \int x(t)\,\text{d} t, \int y(t)\,\text{d} t, \int z(t)\,\text{d} t \right> + \vec{c}\end{align*}$$

for $\vec{r}(t) = \left< x(t), y(t), z(t) \right>$, where $\vec{c}$ is a constant vector. While it’s nice to have the ability to integrate vector-valued functions, we don’t typically find ourselves doing so very often, since vector-valued functions typically represent position, which doesn’t have a very meaningful antiderivative.

If we’re given a vector-valued function that is already a derivative, though, like a velocity function $\vec{v}(t)$, then we can integrate it to find something meaningful: in this case, the position function, up to an integration constant. Let’s collect a bit of what we know about these functions: if $\vec{p}(t)$ gives the position of an object over time, then $\vec{p}'(t) = \vec{v}(t)$ is its velocity, which is tangent to the graph of $\vec{p}(t)$, and $\left| \left| \vec{v}(t) \right| \right|$ is the magnitude of the velocity, also called speed. The second derivative $\vec{p}''(t)$ is the acceleration $\vec{a}(t)$ — also a vector. We can integrate the velocity and acceleration functions to recover the position and velocity ones, respectively, up to a constant.

Arc Length

We can use our newfound ability to differentiate position vectors to compute the length of curves! Let’s look at a particular curve $\vec{r}(t)$ to get a sense of what’s happening.

Here, we can see the position vector in blue and the derivative vector in red. By turning on the points in the Desmos sidebar and increasing $a$, we can see a successive approximation of arc length. At each red point, we take a tangent line to the graph, scaled down so that the total amount the parameter moves across those lines remains constant. For a given number of lines $n$, a line segment tangent to the point $p$ has a tangent line of $\vec{l}(t) = \vec{r}(p) + t\vec{r}'(p)$. If we’re trying to compute the arc length of $\vec{r}(t)$ between $t = a$ and $t = b$, then each of these line segments allows $t$ to move a total of $\frac{b - a}{n}$, the resulting length of the line segment is given by the distance formula, measuring the length of the vector from the start of the line segment to the end. The denominators of $2n$ are due to the line segment being centered at $\vec{r}(p)$: $t$ can only move half of the total amount in each direction.

$$\begin{align*}\left| \left| \left( \vec{r}(p) + \frac{b - a}{2n}\vec{r}'(p) \right) - \left( \vec{r}(p) - \frac{b - a}{2n}\vec{r}'(p) \right) \right| \right| &= \left| \left| \frac{b - a}{n}\vec{r}'(p) \right| \right|\\[4px]&= \frac{b - a}{n} \left| \left| \vec{r}'(p) \right| \right|.\end{align*}$$

For a small number of segments, our approximation is pretty poor, like a too-stringy green bean, but it approaches the graph of the function itself as the number of segments increases. In order to find the total approximation of arc length, we need to add all the segments up as $t$ ranges from $a$ to $b$. Let’s skip the heavy notation for that sum and acknowledge that it’s exactly a Riemann sum! Its limit as the number of segments increases without bound, which is what we’re after anyway, is then given by an integral. We’ve found the following result:

Theorem: arc length

Let $\vec{r}(t)$ be a differentiable vector-valued function with continuous derivatives. The arc length of the graph of $\vec{r}(t)$ between $t = a$ and $t = b$ is

$$\begin{align*}\int_a^b \left| \left| \vec{r}'(t) \right| \right|\,\text{d} t.\end{align*}$$

Exercise: arc length

Find the arc length of the function graphed above — $\vec{r}(t) = \left< \cos(t), \sin(t), \frac{2}{3}t^{3/2} \right>$ — between $t = 0$ and $t = 35$.

The derivative is

$$\begin{align*}\vec{r}'(t) = \left< -\sin(t), \cos(t), t^{1/2} \right>,\end{align*}$$

and its magnitude is

$$\begin{align*}\left| \left| \vec{r}'(t) \right| \right| &= \sqrt{\sin^2(t) + \cos^2(t) + t}\\[4px]&= \sqrt{1 + t}.\end{align*}$$

Integrating that, we can do a $u$-sub with $u = 1 + t$ and $\text{d} u = \text{d} t$ to find

$$\begin{align*}\int_0^{35} \sqrt{1 + t}\,\text{d} t &= \int_0^{35} \sqrt{u}\,\text{d} u\\[4px]&= \left. \left[ \frac{2}{3} u^{3/2} \right] \right|_0^{35}\\[4px]&= \left. \left[ \frac{2}{3} \left( 1 + t \right)^{3/2} \right] \right|_0^{35}\\[4px]&= \frac{2}{3} \left( 36 \right)^{3/2} - \frac{2}{3} \left( 1 \right)^{3/2}\\[4px]&= \frac{2}{3} \left( 216 - 1 \right)\\[4px]&= \frac{430}{3}.\end{align*}$$
