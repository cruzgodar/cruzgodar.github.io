Section 5.10: Lagrange Multipliers

We’re nearly done with multivariable optimization! It’s this section that will largely serve as the bridge between the theoretical foundations we’ve built and the real-world settings they apply to. We’ll open with a twist on the boundary optimization we learned in the last section before developing a new method that will largely supersede it.

We’d often like to optimize a function $f(x, y)$ over a region whose boundary is composed not just of explicit functions like lines. We handled those in the previous section by plugging each function into $f$ to construct various functions of one variable, then optimizing each of those with the techniques of Calculus I. But if the boundary is a more complicated curve like a circle or ellipse — or worse, an implicit equation of $x$ and $y$ where neither variable can be isolated at all — then we have a more difficult task ahead. Let’s take a look at how to handle a problem like this.

Example: a complicated boundary

Let $f(x, y) = xy$ be defined on the astroid $\left\{ (x, y) \in \mathbb{R}^2 \mid x^{2/3} + y^{2/3} \leq 1 \right\}$. Find the absolute maximum and minimum of $f$.

That’s a strange domain! We’ll start by setting $\nabla\! f = \left< y, x \right> = \vec{0}$, which tells us the only critical point of this function is at $(0, 0)$. That’s inside the region, so we do want to consider it; while the second derivative test is unfortunately inconclusive at this point, all we’re trying to do is find the global maximum and minimum, and so we can just record that $f(0, 0) = 0$ and move on.

Now we can focus entirely on the boundary of the region, which is given by $x^{2/3} + y^{2/3} = 1$. It’s not so easy to isolate one variable in terms of another, but we can still write the boundary in terms of a single variable by parameterizing it. That equation looks a lot like a circle: if we can write $x$ and $y$ as functions of $t$ so that $x^{2/3} = \cos^2(t)$ and $y^{2/3} = \sin^2(t)$, then we’ll have defined our region. Solving, we wind up with

$$\begin{align*}x(t) &= \cos^3(t)\\[4px]y(t) &= \sin^3(t),\end{align*}$$

where $0 \leq t \leq 2\pi$.

Now we’re trying to optimize the function

$$\begin{align*}f(x(t), y(t)) &= \cos^3(t)\sin^3(t)\\[4px]&= \left( \cos(t)\sin(t) \right)^3.\end{align*}$$

Written that way, we have a shot at recognizing the interior as $\frac{1}{2}\sin(2t)$, making the whole expression

$$\begin{align*}f(x(t), y(t)) &= \frac{1}{8}\sin^3(2t).\end{align*}$$

To optimize that function, we set $\frac{\mathrm{d}f}{\mathrm{d}t} = 0$ as usual:

$$\begin{align*}\frac{\mathrm{d}}{\mathrm{d}t}\left[ \frac{1}{8}\sin^3(2t) \right] &= 0\\[4px]\frac{3}{8}\sin^2(2t)\cos(2t)(2) &= 0\\[4px]\sin^2(2t)\cos(2t) &= 0\\[4px]\sin(2t) = 0 \text{ or } \cos(2t) &= 0.\end{align*}$$

That’s a complicated way of saying that $2t$ is an integer multiple of $\frac{\pi}{2}$, which is when one of the coordinates on the unit circle is zero. Solving for $t$, we find $t = \frac{k\pi}{4}$ for any integer $k$. Since $0 \leq t \leq 2\pi$, the result is the four coordinate points $(\pm 1, 0)$ and $(0, \pm 1)$ (when $t$ is a multiple of $\frac{pi}{2}$), along with the additional four points

$$\begin{align*}\left( \left( \pm \frac{\sqrt{2}}{2} \right)^3, \left( \pm \frac{\sqrt{2}}{2} \right)^3 \right).\end{align*}$$

Now we can check each of these eight to find the largest and smallest values of $f(x, y)$. Immediately, we can determine that the four points with a coordinate of zero will make $f(x, y) = xy = 0$. Each of the remaining points will have an absolute value of

$$\begin{align*}\left( \frac{\sqrt{2}}{2} \right)^3 \left( \frac{\sqrt{2}}{2} \right)^3 &= \frac{1}{8},\end{align*}$$

with $f(x, y) = \frac{1}{8}$ when the $x$- and $y$-coordinates have the same sign, and $f(x, y) = -\frac{1}{8}$ when they have opposite signs. In total, the points

$$\begin{align*}\left( \left( \frac{\sqrt{2}}{2} \right)^3, \left( \frac{\sqrt{2}}{2} \right)^3 \right), \left( \left( \frac{\sqrt{2}}{2} \right)^3, \left( \frac{\sqrt{2}}{2} \right)^3 \right)\end{align*}$$

are tied for the global maximum, and

$$\begin{align*}\left( \left( -\frac{\sqrt{2}}{2} \right)^3, \left( \frac{\sqrt{2}}{2} \right)^3 \right), \left( \left( \frac{\sqrt{2}}{2} \right)^3, \left( -\frac{\sqrt{2}}{2} \right)^3 \right)\end{align*}$$

are tied for the global minimum.

Exercise: parameterizing a boundary

Find the absolute minimum and maximum of the function $f(x, y) = 2x - 2y$ on the solid circle $x^2 + y^2 \leq 4$ by parameterizing the boundary.

Starting with the critical points, we have

$$\begin{align*}\nabla\! f = \left< 2, -2 \right> = \vec{0},\end{align*}$$

so there are none (which makes some sense; this is a plane, and those don’t have any critical points unless they’re flat). The boundary of our set is given by the equation $x^2 + 4y^2 = 1$, and while we could solve for one variable and plug it into the equation to get a function of one variable, that would result two separate functions to consider because of the $\pm$ that would appear on the square root. Instead, we can turn this into a function of one variable in a different way: parameterizing the boundary. It’s given by

$$\begin{align*}\vec{r}(t) = \left< 2\cos(t), 2\sin(t) \right>, \quad \leq t \leq 2\pi,\end{align*}$$

and so we can optimize the function by plugging these values in for $x$ and $y$ and solving for critical points in $t$:

$$\begin{align*}f\left( \vec{r}(t) \right) &= 2(2\cos(t)) - 2(2\sin(t))\\[4px]&= 4\cos(t) - 4\sin(t)\\[4px]\frac{\mathrm{d}}{\mathrm{d}t} \left[ 4\cos(t) - 4\sin(t) \right] &= -4\sin(t) - 4\cos(t) = 0\\[4px]\sin(t) &= -\cos(t)\\[4px]\tan(t) &= -1\\[4px]t &= \frac{3\pi}{4} \text{ or } t = \frac{7\pi}{4},\end{align*}$$

meaning the critical points of our function on the boundary lie at $\vec{r}\left( \frac{3\pi}{4} \right) = \left( -\sqrt{2}, \sqrt{2} \right)$ and $\vec{r}\left( \frac{7\pi}{4} \right) = \left( \sqrt{2}, -\sqrt{2} \right)$. Checking their values, the first is a minimum and a second one a maximum.

While boundary parameterization was certainly doable in these examples, we’d really like a better approach — it’s not hard to imagine the complexity of a boundary spiraling well beyond our ability to parameterize it, and even these examples weren’t all that easy. With this goal in mind, let’s take a step back and approach the challenge of boundary parameterization from a different, slightly more applied perspective.

We opened the last section by thinking about a function $f(x, y) = 2x + xy + 5y$ giving the monthly profit to a birding store selling $x$ pairs of binoculars and $y$ birdhouses per month, subject to the constraint $10x + 5y \leq 100$. To get at a different method of optimization that doesn’t require boundary parameterization, let’s think about level curves. The maximum $(a, b)$ that we’re looking for will occur at a point on some level curve of $f$, and also on the line $10x + 5y = 100$; let’s draw both.

Immediately, we can see that we need to pick a small enough value of $c$ that the level curve $2x + xy + 5y = c$ actually intersects $10x + 5y = 100$; otherwise, there won’t be any point in the intersection. But on the other hand, we want $c$ to be as large as possible, because crucially, $c$ is the output of $f(x, y)$ — it’s the thing we’re trying to maximize. We therefore are looking for a value of $c$ where the level curve just barely touches the line $10x + 5y = 100$, or in other words, where the two are tangent.

Finding that $c$-value might be easy if the boundary is a line, but what if it’s a more complicated curve? We owe it to ourselves to find a more general solution to finding a $c$-value whose level curve is tangent to the boundary.

Let’s begin by giving the function defining the constraint a name: $g(x, y)$. And to handle more general boundary conditions, let’s just insist that the boundary is given by $g(x, y) = 0$, so in this example, $g(x, y) = 10x + 5y - 100$. The magic comes from writing $g$ as a function of multiple variables, so that we can think of $g(x, y) = 0$ as being a level curve of $g$. Then these two curves being tangent is equivalent to saying that we can move in the same direction and not increase in height on either the graph of $f(x, y)$ or $g(x, y)$ — in other words, that there’s a single direction orthogonal to both $\nabla\! f$ and $\nabla\! g$. That happens when $\nabla\! f$ and $\nabla\! g$ point in the same direction or in opposite directions, and we can account for both cases by simply saying that $\nabla\! f$ is a constant multiple (possibly negative) of $\nabla\! g$. In the typical notation of this method, we’ll denote that constant multiple by the Greek letter $\lambda$ (pronounced “lambda”), and call is a Lagrange multiplier when we occasionally have need for a name.

Let’s see how this applies to our example. Given the profit function $f(x, y)$ that we’re trying to maximize and the constraint $g(x, y)$, we set up two equations:

$$\begin{align*}\nabla\! f(x, y) &= \lambda \nabla\! g(x, y)\\[4px]g(x, y) &= 0\end{align*}$$

While this might seem like two equations and three unknowns, it’s not! Let’s expand it out and see why.

$$\begin{align*}\left< 2 + y, x + 5 \right> &= \lambda \left< 10, 5 \right>\\[4px]10x + 5y - 100 &= 0.\end{align*}$$

The two equations are actually three, since in the first one, both the first and second components of the vector must be equal. Therefore,

$$\begin{align*}2 + y &= 10\lambda\\[4px]x + 5 &= 5\lambda\\[4px]10x + 5y - 100 &= 0.\end{align*}$$

Plugging into the third equation,

$$\begin{align*}10(5\lambda - 5) + 5(10\lambda - 2) - 100 &= 0\\[4px]50\lambda - 50 + 50\lambda - 10 - 100 &= 0\\[4px]100\lambda &= 160\\[4px]\lambda &= \frac{8}{5}.\end{align*}$$

Returning to solve for $x$ and $y$,

$$\begin{align*}x &= 3\\[4px]y &= 14.\end{align*}$$

Our maximum is therefore at $(3, 14)$, and the output is $f(3, 14) = 118$. Although it’s an oversimplified example, this method genuinely gave us an output to inform how we should allocate our spending on products to maximize profit!

Lagrange multipliers are borderline magic. They let us optimize functions on a boundary without ever needing to parameterize a thing, and they don’t just work for finding maxima — the same logic tells us that we can minimize $f$ on the boundary in this way too. Let’s state the method carefully.

Method: Lagrange multipliers

Let $f(x, y)$ be a function of two variables, and let $g(x, y)$ be any function. To optimize $f$ on the curve $g(x, y) = 0$,

1. Solve the system of equations

$$\begin{align*}\nabla\! f(x, y) &= \lambda \nabla\! g(x, y)\\[4px]g(x, y) &= 0.\end{align*}$$

2. Of all the solutions $(x, y)$ in step 1, the largest and smallest values of $f(x, y)$ among those solutions are the values we’re looking for. If there is only a single solution $(a, b)$, then it’s either a maximum or minimum. To determine which, just plug in another point $(c, d)$ with $g(c, d) = 0$ to see if $f(c, d)$ is larger or smaller than $f(a, b)$.

Let’s now return to the astroid example to see the improvement.

Exercise: Lagrange multipliers

Let $f(x, y) = xy$ be defined on the astroid $\left\{ (x, y) \in \mathbb{R}^2 \mid x^{2/3} + y^{2/3} = 1 \right\}$ (i.e. just on the boundary this time). Find the absolute maximum and minimum of $f$.

Instead of parameterizing the boundary in terms of $t$, let’s use Lagrange multipliers. We have $f(x, y) = xy$ and $g(x, y) = x^{2/3} + y^{2/3} - 1$, and so our system of equations is

$$\begin{align*}\nabla\! f(x, y) &= \lambda \nabla\! g(x, y)\\[4px]g(x, y) &= 0\\[4px]~\\[4px]\left< y, x \right> &= \lambda\left< \frac{2}{3}x^{-1/3}, \frac{2}{3}y^{-1/3} \right>\\[4px]x^{2/3} + y^{2/3} - 1 &= 0\\[4px]~\\[4px]y &= \frac{2}{3}\lambda x^{-1/3}\\[4px]x &= \frac{2}{3}\lambda y^{-1/3}\\[4px]x^{2/3} + y^{2/3} - 1 &= 0.\end{align*}$$

It’s often easiest to just eliminate $\lambda$ from complicated systems like this:

$$\begin{align*}\frac{3}{2} y x^{1/3} &= \lambda\\[4px]\frac{3}{2} x y^{1/3} &= \lambda\\[4px]\frac{3}{2} y x^{1/3} &= \frac{3}{2} x y^{1/3}\\[4px]x^{2/3} &= y^{2/3}\end{align*}$$

While we actually can factor this, let’s just use the quadratic formula — it’s not amazing, but it certainly gets the job done.

$$\begin{align*}x &= \frac{-\frac{2}{\sqrt{3}}y \pm \sqrt{\frac{4}{3}y^2 + 4y^2}}{2}\\[4px]&= \frac{-\frac{2}{\sqrt{3}}y \pm \sqrt{\frac{16}{3}y^2}}{2}\\[4px]&= \frac{-\frac{2}{\sqrt{3}}y \pm \frac{4}{\sqrt{3}}y}{2}\\[4px]&= -\frac{1}{\sqrt{3}}y \pm \frac{2}{\sqrt{3}}y,\end{align*}$$

so either $x = -\frac{3}{\sqrt{3}}y = -\sqrt{3}y$ or $x = \frac{1}{\sqrt{3}}y$. And now we can plug this back into $x^2 + y^2 = 1$! The first equation gives

$$\begin{align*}\left( -\sqrt{3} y\right)^2 + y^2 &= 1\\[4px]3y^2 + y^2 &= 1\\[4px]y &= \pm \sqrt{\frac{1}{4}} = \pm \frac{1}{2},\end{align*}$$

so we get the points $\left( -\frac{\sqrt{3}}{2}, \frac{1}{2} \right)$ and $\left( \frac{\sqrt{3}}{2}, -\frac{1}{2} \right)$. When $x = \frac{1}{\sqrt{3}}y$,

$$\begin{align*}\left( \frac{1}{\sqrt{3}} y\right)^2 + y^2 &= 1\\[4px]\frac{1}{3}y^2 + y^2 &= 1\\[4px]y &= \pm \sqrt{\frac{3}{4}} = \pm \frac{\sqrt{3}}{2},\end{align*}$$

which gives us $\left( \frac{1}{2}, \frac{\sqrt{3}}{2} \right)$ and $\left( -\frac{1}{2}, -\frac{\sqrt{3}}{2} \right)$.

To wrap up the problem, we just need to plug all four of these into the original function:

$$\begin{align*}f\left( -\frac{\sqrt{3}}{2}, \frac{1}{2} \right) = f\left( \frac{\sqrt{3}}{2}, -\frac{1}{2} \right) &= \frac{3}{2}\\[4px]f\left( \frac{1}{2}, \frac{\sqrt{3}}{2} \right) = f\left( -\frac{1}{2}, -\frac{\sqrt{3}}{2} \right) &= \frac{7}{2}.\end{align*}$$

The end result is that $f$ has minima on this boundary at the first two points and maxima at the second two points.
