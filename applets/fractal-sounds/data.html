Fractal Sounds

Half the reason I made the sorting algorithms applet was to get practice using the WebAudio API so that I could make this. Iteration-based fractals like the Mandelbrot set are generated by iterating points over and over, and seeing if the path they take remains bounded or not. Some bounded points converge to a single value, but others get trapped in loops, repeating the same values over and over. For those, we can convert the $x$- and $y$-coordinates into the left and right channels of an audio track by interpolating them into sine waves. If everything is managed just right, we have a way to create sounds from fractals. This idea comes directly from CodeParade, who wrote this as a native application — while the WebAudio API is limited in when sounds can be played, I think there’s something to be gained in a web-based implementation. If at all possible, please use headphones with this applet!

Controls

Drag on the scene with the mouse or two fingers to look around, and pinch or scroll to zoom. Click or tap to play a sound — the colors roughly indicate the fundamental frequencies of each point.
